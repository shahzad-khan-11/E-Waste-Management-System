# E-Waste Management System

The E-Waste Management System is a digital platform designed to manage the safe collection, tracking, and recycling of electronic waste such as old computers, mobile phones, batteries, and other electronic devices.  
It helps users request pickup services, view nearby recycling centers with live location support, access emergency assistance, and earn reward points for responsible e-waste disposal.
